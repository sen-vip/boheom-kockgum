# 콕검 · 보험서류 검토 보조 웹앱

**콕검**은 학교 행정실 계약담당자가 보험서류에서 확인해야 할 부분을 빠르게 찾고 정리할 수 있도록 돕는 로컬 웹앱입니다.

> 파일은 내 PC에, 검토 포인트는 한눈에.

## 핵심 원칙

- 서버 전송 없음
- 브라우저 내부 분석
- 원본 파일 자동 저장 없음
- 결과는 검토 보조용
- 최종 확인은 담당자가 원본 서류에서 직접 진행

## 기능

- PDF, JPG, PNG 여러 파일 업로드
- 파일명과 무관하게 내용 기준으로 분석
- 화재보험, 영업배상책임보험, 생산물배상책임보험, 여행자보험 관련 문구 확인
- 보험기간, 업체명 후보, 주소 후보 확인
- 확인됨 / 확인 필요 / 누락 의심 / 분석 어려움 상태 표시
- 검토 의견 자동 작성
- TXT 저장, 결과 전체 복사, CSV 검토표 다운로드

## 지원 파일 및 권장 제한

- 지원 형식: PDF, JPG, PNG
- 한 번에 최대 10개 파일
- 파일 1개당 10MB 이하 권장
- 전체 50MB 이하 권장
- PDF 1개당 20쪽 이하 권장

## 여행자보험 주의사항

여행자보험은 학교마다 보상한도 기준이 다를 수 있습니다.

콕검은 다음 항목의 존재 여부만 확인합니다.

- 여행자보험 관련 문구
- 보험증권
- 보험 약관
- 보험료 납부 영수증
- 개별 피보험자 보험료 표시 여부
- 상해 사망·후유장애 항목
- 상해 치료실비 항목
- 질병 사망·후유장애 항목
- 질병 치료실비 항목
- 배상책임 항목
- 휴대품 항목

금액의 적정성은 원본 서류와 계약조건을 기준으로 담당자가 직접 확인해야 합니다.
